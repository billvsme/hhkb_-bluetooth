# hhkb_bluetooth
hhkb 蓝牙改造项目，本着尽量简单，少改动，少焊接，可以恢复的原则，提供2种方案。方法一其实就是一个usb转蓝牙模块，方法二真正的改造，需要拆机和焊接

## 方案一
本质就是一个usb转蓝牙模块  
**材料**  

1. arduino uno
2. usb host shield 2.0
3. XM-04-HID-K (购买时候请搜索XM-04-HID-S，然后询问卖家需要XM-04-HID-K😂, 并且问他文档)
